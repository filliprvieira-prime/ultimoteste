
# Atlantic Garden - Landing Page

Este projeto está pronto para ser publicado no GitHub.

## Como publicar:
1. Use a ferramenta de Git/GitHub no menu lateral do seu editor.
2. Conecte sua conta e publique o repositório como "Public".
3. Nas configurações do repositório no GitHub, vá em **Settings > Pages**.
4. Selecione **GitHub Actions** como fonte do build.
5. O site estará online em instantes!
